   - Otherwise: return diff list joined by `'; '`.

6. Use the Unicode arrow character `→` (U+2192), NOT `->`. Tests will bind to this exact char.

Do NOT add any DB imports, any `import 'server-only'`, any logging, any side effects. The file is pure and importable from both server and client (though we only use it server-side).
