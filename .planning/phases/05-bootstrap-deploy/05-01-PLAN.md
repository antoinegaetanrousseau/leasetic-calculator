| Boundary | Description |
|----------|-------------|
| Public internet → Next.js app | Once deployed in plan 07, anyone can hit the production URL. The placeholder page in this plan does not yet expose any data, but the build artifacts and env-var contract established here propagate to production. |
| Build artifact → runtime | `.next/standalone/server.js` is the OVH-portable runtime artifact. Anything baked in at build time (env vars marked NEXT_PUBLIC_, build-time DB queries via static rendering) ships to clients. |

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
